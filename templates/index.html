Linear regression models the relationship between a dependent variable (Price) and an independent variable (Size/Area) by fitting a linear equation to observed data.

Standard Form:
y = m · x + c
y (Dependent Variable): Predicted Price ($)
x (Independent Variable): Property Area (sq ft)
m (Slope / Coefficient): Price change per sq ft
c (Y-Intercept): Base valuation value when Area is zero
